# Agent Feature Ideas (ESP32 + Android Companion)

This document captures practical voice-agent features for daily use.

## Core Daily Features (with why each helps)

1. **Morning Briefing**  
   Gives weather, first meetings, commute, and top tasks in one short update.
2. **"What's Next?" Query**  
   Instantly answers your next commitment without opening calendar apps.
3. **Calendar Readout by Context**  
   Supports natural queries like "after lunch" or "this afternoon."
4. **One-Shot Event Creation**  
   Converts spoken requests into calendar events quickly.
5. **Conflict Detection Before Booking**  
   Prevents double-booking and suggests free alternatives.
6. **Smart Reminders from Speech**  
   Captures reminders in the moment so fewer tasks are forgotten.
7. **Quick Capture Inbox**  
   Stores ideas and to-dos fast, reducing mental overhead.
8. **Follow-Up Assistant**  
   Reminds you to follow up when responses are overdue.
9. **Meeting Prep Snapshot**  
   Surfaces agenda and relevant context before a meeting starts.
10. **Post-Meeting Action Extraction**  
    Turns spoken summaries into actionable tasks and deadlines.
11. **Focused Work Timer**  
    Starts focus blocks and helps reduce interruptions.
12. **Daily Planning Prompt**  
    Encourages selecting top priorities early in the day.
13. **Evening Wrap-Up**  
    Reviews unfinished items and sets the first step for tomorrow.
14. **Live Web Lookup Tool**  
    Answers current factual questions with up-to-date information.
15. **Travel-Aware Alerts**  
    Uses traffic-aware reminders to reduce lateness.
16. **Routine Nudges**  
    Supports consistency for habits like water, meds, and breaks.
17. **Shopping List by Voice**  
    Lets you add/check items hands-free at the moment you think of them.
18. **Safe Action Confirmations**  
    Prevents accidental sends/cancellations/bookings via confirmation.
